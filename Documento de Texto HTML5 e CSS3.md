## **HTML5**



#### **Tags**

 	Para sinais de <> use \&lt; e \&gt;

 	Para um termo em destaque use <strong>

 	Para um termo em itálico use <em>

 	Para marcadores de texto use <mark>

 	Para marcar um texto como excluído <del>

 	Para marcar um texto como inserido depois use <ins>

 	Para números fracionários a cima use <sup> e <sub> para baixo

 	Para códigos escritos use <pre> e <code>

 	Para citações simples use <q>

 	Para citações completas use <blockquote>

 	Para abreviações use <abbr>

 	Para lista de definições use <dl> <dt> (termo) e <dd> (descrição)

 	Para personalização de listas ordenadas use type="1 A a I i"

 	Para personalização de listas não ordenadas use	type="disc circle square"

 	Para links use target="\_blank" ou "\_self" e também rel="external, prev ou next"

 	Para imagens dinâmicas use <picture> <source media type>

 	Para reproduzir áudio use <áudio preload autoplay controls loop> e <source src type>

 	Para adicionar capa ao vídeo use "poster"

 	Para vários formatos de vídeos use <vídeo controls loop autoplay> <source src type>





##### Ícone

 	Para escolher um favicon para seu site, utilize o flaticon ou o icon archive
